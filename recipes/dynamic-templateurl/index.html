Use a dynamic templateUrl

by Sascha Brink

Buy this book

Problem

You have several html templates that you want to use dynamically inside your directive.

Solution

Inside a directive, templateUrl can not only take a string but also a function. The function has the element and the attributes as parameters.

In the following example, we dynamically fill a textarea with two different templates that are loaded dynamically. We write a directive, which is named ‘prefill’. This takes the name of the template as argument.

<html ng-app="cookbookApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.0/angular.min.js"></script>
  <script src="application.js"></script>
</head>
<body>
  <textarea prefill="filler1"></textarea>
  <textarea prefill="filler2"></textarea>
</body>
</html>

The directive itself consists only of the templateUrl with a function. To create the url, here we just take the argument from the prefill attribute and append .html to it.

angular.module('cookbookApp', [])
  .directive('prefill', function() {
    return {
      templateUrl: function(element, attrs) {
        return attrs.prefill + '.html';
      }
    };
  });
