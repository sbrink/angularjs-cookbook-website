Submit form on enter key

by Sascha Brink

Buy this book

Problem

If you press enter in a form field, the form usually is submitted to the action defined in the ´´ tag. But you want to call a function in your controller instead.

Solution

The solution is to leave out the action attribute in <form> and use ng-submit. When you press return inside a form field, the expression in ng-submit is executed.

A complete example:

<html ng-app="cookbookApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.0/angular.min.js"></script>
  <script src="application.js"></script>
  <link rel="stylesheet" ng-href="style.css"/>
</head>
<body ng-controller="MainController">
  <form name="form" ng-submit="addTask(taskName)" novalidate>
    <input type="text" ng-model="taskName" />
  </form>

  <ul>
    <li ng-repeat="task in tasks track by $index" ng-bind="task"></li>
  </ul>
</body>
</html>

angular.module('cookbookApp', [])
  .controller('MainController', function($scope) {
    $scope.tasks = [];

    $scope.addTask = function(taskName) {
      $scope.tasks.push(taskName);
    };
  });

We use track by $index in ng-repeat here. This is a hack to allow duplicate entries in your list. Learn more about it here
